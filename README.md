# Banking Customer Behavior & Predictive Analytics

This project utilizes Python and Machine Learning to analyze banking customer data, moving beyond descriptive SQL queries into predictive modeling. The goal is to understand customer demographics and predict financial behaviors.

## 📊 Project Scope
As an expansion of database management projects, this notebook performs end-to-end data science tasks:
* **Data Wrangling:** Cleaning and preprocessing raw banking records (Customer, Account, and Branch data).
* **Exploratory Data Analysis (EDA):** Visualizing distributions of account balances, occupations, and geographical trends.
* **Predictive Modeling:** Implementing classification algorithms to identify high-value customers or segment users for targeted product management.

## 💻 Tech Stack
* **Language:** Python 3.x
* **Libraries:** * `pandas` & `numpy`: Data manipulation
    * `matplotlib` & `seaborn`: Statistical visualizations
    * `scikit-learn`: Machine Learning (Preprocessing, Model Selection, Metrics)

## 📈 Key Insights
1. **Occupation Trends:** Identified which professional segments maintain the highest average current balances.
2. **Geographic Concentration:** Analysis of branch performance across different cities.
3. **Behavioral Prediction:** Classification of customers based on account tenure and transaction frequency.

## 🚀 How to Run
1. Clone the repository.
2. Ensure you have the required libraries installed: `pip install pandas numpy seaborn scikit-learn`.
3. Open `Group Project4 (3).ipynb` in Jupyter Notebook or Google Colab.
4. Run all cells to reproduce the analysis and model results.
